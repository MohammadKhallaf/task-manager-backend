## Instructions 
1. install dependencies
   1. server --> express
   2. db --> mongoose
   3. configs 
      1.  environment -> dotenv
      2. encrypt -> bcrypt
      3. jwt -> jsonwebtoken
      4. server -> cors | helmet
```bash
npm install express mongoose bcryptjs jsonwebtoken dotenv helmet cors
```
